# ⚙️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/vrutigadhiya/portfolio.git
```

### Navigate to Project

```bash
cd portfolio
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
